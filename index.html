Virtual Currency Manager

הורדה עצמית של תוסף ה-Chrome + הוראות התקנה.

⬇️ הורד/י את ההרחבה (ZIP)

התקנה (Chrome)

  1. הורדה: לחצו על הכפתור ושמרו את ה-ZIP.
  2. פתחו chrome://extensions וסמנו Developer mode.
  3. חלצו את ה-ZIP לתיקייה.
  4. לחצו Load unpacked ובחרו את התיקייה שחילצתם (שבה נמצא manifest.json).

אם תיתקלו בשגיאת 429 בזמן היסטוריית מחירים – זה קצב הבקשות של CoinGecko. המתינו ~30 שניות ונסו שוב.

קוד מקור

הקוד המלא נמצא בתוך ה-ZIP. אפשר לראות קוד גם דרך DevTools → Sources (popup.js, service.js, dao.js).
